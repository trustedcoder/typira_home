# Typira Home

Welcome to the Typira monorepo. Typira is an intelligent mobile typing assistant that integrates directly into your custom keyboard to provide real-time AI insights, corrections, and productivity enhancements.

## Project Structure

This repository is organized into the following main components:

- **Mobile Application**: Located in `mobile/typira`. This is the user-facing mobile app and custom keyboard built with **Flutter**.
- **Backend API**: Located in `backend_api/typira`. This is the server-side API built with **Flask** that powers the app's data syncing, authentication, and advanced AI processing.
